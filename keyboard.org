** 快捷键一览表
   | 热键         | 功能                 | 备注     |
   |--------------+----------------------+----------|
   | C- a         | 全选                 |          |
   | M-S- f       | 向前选择一个词       |          |
   | M-S- b       | 向后选择一个词       |          |
   | M-S- e       | 向前选择一个段落     |          |
   | M-S- a       | 向后选择一个段落     |          |
   | <Mouse-2>    | 选择到光标所在区域   | 鼠标右键 |
   | S- <Mouse-1> | 选择到光标所在区域   | 鼠标左键 |
   |--------------+----------------------+----------|
   | M-S- <       | 移动光标到头部       |          |
   | M-S- >       | 移动光标到尾部       |          |
   | M-S- {       | 移动光标到前一个段落 |          |
   | M-S- }       | 移动光标到后一个段落 |          |
   |--------------+----------------------+----------|
   |              |                      |          |
